# Computer Vision Coursework – ITAI 1378  
This repository contains my coursework, assignments, and experiments for **ITAI 1378 – Introduction to Artificial Intelligence (Computer Vision Module)**.  
It includes hands‑on exercises in image processing, feature extraction, edge detection, annotation, and applied computer vision techniques using Python, OpenCV, NumPy, and Matplotlib.

---

## 📚 Course Overview
This course introduces foundational concepts in computer vision, including:
- Image representation and color spaces  
- Image preprocessing and enhancement  
- Edge detection and segmentation  
- Feature extraction  
- Object detection fundamentals  
- Real‑world applications of computer vision  
- Ethical considerations in AI and imaging systems  

All code in this repository is written in Python and executed in Jupyter/Colab environments.

---

## 🧪 Contents of This Repository

### **1. Image Processing Basics**
- Loading and displaying images  
- Converting between BGR and RGB  
- Grayscale conversion  
- Cropping and slicing images  
- Drawing shapes and annotations (rectangles, text overlays)

### **2. Edge Detection & Filters**
- Canny edge detection with multiple threshold pairs  
- Sobel and Laplacian filters  
- Comparison of edge detection techniques  
- Visualizing differences between thresholds and methods  

### **3. Image Enhancement**
- Brightening and contrast adjustments  
- Gamma correction  
- Using `cv2.add`, NumPy operations, and clipping  
- One‑copy safe workflow for reproducible experiments  

### **4. Real‑World Computer Vision Application Analysis**
A written analysis exploring:
- Medical image analysis  
- How computer vision is used in clinical diagnostics  
- Technologies behind it (CNNs, ViTs, imaging hardware)  
- Benefits, challenges, and ethical considerations  
- Future impact on healthcare  

### **5. Additional Experiments**
- Annotating custom images  
- Saving processed outputs  
- Working with personal photos (BGR→RGB conversion)  
- Exploratory tasks and stretch goals from class
